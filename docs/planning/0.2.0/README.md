# 0.2.0 规划文档

本目录保存 `0.2.0` 的规划文档和落地边界。

当前 `0.2.0` 已按个人使用标准完成核心功能落地；这里的文档保留为后续回看范围和取舍的依据。

## 文档清单

- [requirements.md](./requirements.md)：0.2.0 产品需求草案。
- [detailed-design.md](./detailed-design.md)：0.2.0 详细设计文档。
- [implementation-tasks.md](./implementation-tasks.md)：0.2.0 最小可执行任务拆分。
- [prompt.md](./prompt.md)：0.2.0 vibe coding 执行提示词归档。

## 规划重点

- 核心目标：从“按游戏查看比赛”升级为“按条件查询赛事与结果”。
- 增加赛事信息多维筛选，不再只按游戏项目筛选。
- 支持历史日期和自定义日期范围，方便查询过往比赛结果。
- 区分赛程和赛果视图。
- 增加赛事、队伍、状态、赛区、关键词等筛选能力。
- 增加比赛详情展开或详情弹窗，展示 BO、比分、状态、数据来源和更新时间。
- 收藏关注、热门赛事和我的关注标签暂缓到后续版本，避免 0.2.0 范围过大。
